---
title: Create Form
api: POST /forms
description: Create a new form from a natural language description
---

Create a new form programmatically by providing a natural language description. Our AI will automatically infer the appropriate form fields, types, and structure based on your prompt.

<ParamField body="description" type="string" required>
  Natural language description of the form and its fields.
  Example: "Create a contact form with Name, Email, Message"
</ParamField>

<ParamField body="title" type="string">
  Optional explicit form title. Overrides the title inferred from description.
</ParamField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://quipform.com/api/v1/forms \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "description": "Create a contact form with Name, Email, and Message",
    "title": "Customer Inquiry"
  }'
```

```javascript Node.js
fetch('https://quipform.com/api/v1/forms', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    description: 'Create a contact form with Name, Email, and Message',
    title: 'Customer Inquiry'
  })
})
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
</RequestExample>
