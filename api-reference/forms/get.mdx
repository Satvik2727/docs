---
title: Get Form
api: GET /forms/{id}
description: Get a single form by ID
---

Retrieve the full schema and metadata for a specific form by its unique identifier. This includes all configured fields, which is necessary if you intend to map submission answers back to their respective questions.

<ParamField path="id" type="string" required>
  The form UUID
</ParamField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://quipform.com/api/v1/forms/abc123 \
  --header 'Authorization: Bearer <token>'
```

```javascript Node.js
fetch('https://quipform.com/api/v1/forms/abc123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
})
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
</RequestExample>

<ResponseExample>

{
  "id": "abc123",
  "title": "Contact Form",
  "slug": "contact-form-abc123",

</ResponseExample>
