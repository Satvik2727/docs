---
title: List Forms
api: GET /forms
description: Returns your most recent forms
---

Use this endpoint to retrieve a paginated list of all forms associated with your account. Forms are returned in reverse chronological order based on creation date. This is useful for building dashboard views or syncing form metadata to your internal database.

<ParamField query="limit" type="number" default="20">
  Max number of forms to return (max 100)
</ParamField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://quipform.com/api/v1/forms \
  --header 'Authorization: Bearer <token>'
```

```javascript Node.js
fetch('https://quipform.com/api/v1/forms', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
})
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
</RequestExample>

</ResponseExample>
